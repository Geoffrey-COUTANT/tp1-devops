# API Ping
## Description
API de test pour le projet. Cette API permet de vérifier la connectivité un port modifiable (8000 par défaut) et la disponibilité du service.

# Lancer le projet

## Installation des dépendances :

```
npm install
```
ou
```
yarn install
```

## Lancement du serveur web :

```
npx tsx index.ts
```
